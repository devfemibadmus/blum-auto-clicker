
# $BLUM Auto Clicker Tool

A Python-based Telegram Desktop auto clicker.

## Getting Started

### Clone the repo
```
git clone https://github.com/yourusername/your-repo-name.git
cd your-repo-name
```

### Install dependencies
```
pip install -r requirements.txt
```

### Run the tool
```
python main.py
```
